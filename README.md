# Golang CRUD API – Kasir API  
### Bootcamp Golang with Mas Umam

RESTful CRUD API sederhana menggunakan **Golang**, **Gin**, **GORM**, dan **PostgreSQL**, dengan arsitektur **Idiomatic Go (feature-based / clean architecture)**.  
Project ini dideploy menggunakan **Railway** dan ditujukan sebagai contoh backend API yang rapi, scalable, dan production-ready.

---

## Live Demo (Production)

Base URL: https://golangexamplecrudapi-production.up.railway.app


---

## Features

- CRUD Category (Create, Read, Update, Delete)
- RESTful API
- PostgreSQL database
- Gin Web Framework
- GORM ORM
- Clean & Idiomatic Go architecture
- Ready for deployment (Railway)

---

## API Endpoints — Categories

### Get All Categories
**GET**
/api/categories

### Get Category by ID
**GET**
/api/categories/{id}

### Create Category
**POST**
/api/categories

### Update Category by ID
**PUT**
/api/categories/{id}

### Delete Category by ID
**DELETE**
/api/categories/{id}
